# Amazon UI Clone

This project is a **front-end clone of the Amazon homepage UI** built using **HTML and CSS**.
The goal of this project was to practice **layout design, Flexbox, styling, and responsive UI development**.

## 🚀 Features

* Amazon-like **navigation bar**
* **Search bar UI**
* **Product sections**
* **Footer layout**
* Clean **CSS styling**
* Use of **Flexbox for layout**

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* Flexbox

## 📂 Project Structure

```
Amazon-UI-clone
│
├── index.html
├── style.css
├── favicon.ico
└── images
```

## 🎯 Purpose of the Project

This project was created to:

* Practice **HTML page structure**
* Improve **CSS layout skills**
* Understand how large websites structure their **UI components**
